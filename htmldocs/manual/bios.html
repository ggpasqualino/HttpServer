Como atualizar o BIOS

Dell™ Studio XPS™ 1340 Manual de serviço

  Como atualizar o BIOS usando um CD

  Como gravar no BIOS a partir da unidade de disco rígido


Se um CD de atualização do BIOS tiver sido fornecido com a nova placa de sistema, atualize o BIOS a partir do CD. Se você não tiver um CD de atualização do BIOS, atualize o BIOS a partir da unidade de disco rígido.

Como atualizar o BIOS usando um CD

  1. Certifique-se de que o adaptador CA esteja conectado e que a bateria principal esteja instalada corretamente.

NOTA: se você atualizar o BIOS a partir do CD de atualização, configure o computador para inicializar a partir do CD antes de inseri-lo na unidade.
  1. Insira o CD de atualização do BIOS e reinicie o computador.

Siga as instruções exibidas na tela. O computador continuará a inicializar e atualizará o novo BIOS. Quando a atualização estiver concluída, o computador reinicializará automaticamente.

  1. Pressione <F2> durante o POST para entrar no programa de configuração do sistema.

  2. Selecione o menu Exit (Sair).

  3. Selecione Load Setup Defaults (Carregar configurações padrão) para restaurar as configurações padrão do computador.

  4. Remova o CD de atualização do BIOS da unidade.

  5. Selecione Exit Saving Changes (Sair e salvar alterações) para salvar as alterações na configuração.

  6. A atualização do BIOS é concluída, e o computador é reiniciado automaticamente.

Como gravar no BIOS a partir da unidade de disco rígido

  1. Certifique-se de que o adaptador CA esteja conectado e que a bateria principal esteja instalada corretamente.

  2. Ligue o computador.

  3. Localize o arquivo mais recente de atualização do BIOS do seu computador no site support.dell.com (em inglês).

  4. Clique em Download Now (Baixar agora) para baixar o arquivo.

  5. Se a janela Export Compliance Disclaimer (Exoneração de responsabilidade sobre compatibilidade com as leis de exportação) for exibida, clique em Yes, I Accept this Agreement (Sim, eu aceito este contrato).

A janela File Download (Download de arquivo) é exibida.

  1. Clique em Save this program to disk (Salvar este programa no disco) e clique em OK.

A janela Save In (Salvar em) é exibida.

  1. Clique na seta para baixo para exibir o menu Save In (Salvar em). Selecione Desktop (Área de trabalho) e clique em Save (Salvar).

O arquivo será salvo na sua área de trabalho.

  1. Clique em Close (Fechar) se a janela Download Complete (Download concluído) for exibida.

O ícone do arquivo é exibido na área de trabalho e possui o mesmo nome que o arquivo de atualização do BIOS baixado.

  1. Clique duas vezes no ícone do arquivo mostrado na área de trabalho e siga as instruções apresentadas na tela.
